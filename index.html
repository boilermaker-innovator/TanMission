Implementation Notes

To implement this flow with Raisly:

  1. Create Raisly Campaigns - Set up separate campaigns for each project (Well, Tractors, Training, Support)
  2. Configure donation amounts - Set suggested amounts that align with your "$5 per unit" concept
  3. Get embed codes - Raisly provides embed codes for each campaign
  4. Integrate with your üëçi popups - Load the appropriate Raisly embed when a project is selected

This approach gives you the best of both worlds - your engaging üëçi interface with Raisly's proven donation processing.
